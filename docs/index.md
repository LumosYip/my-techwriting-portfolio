# Lumos's Docs

Welcome to **Lumos's Docs**, a personal documentation portal for Technical Writing, Translation, and Digital Forensics.

This site contains:

- **Technical Writing** – Guidelines, workflow, and best practices for writing professional documentation.
- **Translation** – Notes on CAT tools, localization tips, and translation workflows.
- **Forensics** – (Coming soon) Guides and resources for digital forensics.
- **Tools** – (Coming soon) Recommended software and tutorials for documentation and technical work.

## How to navigate

Use the left-hand sidebar to explore each category. Click on a page title to read the content.  
All documents are written in **Markdown** and generated into a clean, structured website using **MkDocs**.
